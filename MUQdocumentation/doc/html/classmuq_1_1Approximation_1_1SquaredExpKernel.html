Detailed Description

This class implements a kernel of the form

\[ k(x,y) = \sigma^2 \exp\left(-\frac{1}{2}\frac{|x-y|^2}{L^2}\right) \]

for some variance \(\sigma^2\) and lengthscale \(L\).
